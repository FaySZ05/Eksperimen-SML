# Eksperimen-SML
This project focuses on the end-to-end processing of Bitcoin historical data, beginning with automated dataset acquisition via kagglehub and moving into a rigorous Exploratory Data Analysis (EDA) phase. The pipeline transforms raw Unix timestamps into a structured time-series index and performs a thorough structural audit to handle missing values and duplicates. Key financial features, including OHLC prices and trading volume, are analyzed through descriptive statistics and correlation matrices to identify market patterns and ensure data integrity before any predictive modeling takes place.

To ensure the dataset is optimized for machine learning, a robust preprocessing workflow is implemented, featuring feature standardization and advanced outlier management. Using StandardScaler, the data is normalized to achieve a zero mean and unit variance, while outliers are addressed through a Z-score capping method to preserve time-series continuity without losing critical information. The process is finalized with comprehensive visualizations, including distribution histograms and heatmaps, providing a clear understanding of feature relationships and preparing the data for seamless integration into model training or CI/CD deployment pipelines.
